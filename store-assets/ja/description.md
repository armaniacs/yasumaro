# Yasumaro - AI ブラウジングロガー

Yasumaro は、AI による要約付きであなた専用のブラウジングデータベースを自動構築します。

**できること:**
- 実際に読んだページを自動保存（滞在時間とスクロール深度で判定）
- お手持ちの API キーで AI 要約（Gemini、OpenAI、Groq、ローカル LLM 対応）
- すべてをローカルの SQLite に保存 — 外部サーバー不要
- 必要に応じて Obsidian デイリーノートへ同期

**プライバシー最重視:**
閲覧履歴はすべてあなたのデバイスに残ります。データはあなたのものです。

**こんな方に:**
- 記事を収集する研究者
- 新技術を学ぶ開発者
- 業界ニュースを追うビジネスパーソン
- 知識ベースを作る学生
