# Store Assets Workflow

## Directory Structure

```
store-assets/
├── en/
│   └── description.md
├── ja/
│   └── description.md
├── screenshots/
│   ├── popup-light.png
│   ├── popup-dark.png
│   ├── dashboard-history.png
│   └── dashboard-settings.png
└── promo/
    └── video.mp4 (optional)
```

## Release Checklist

- [ ] Update screenshots to reflect latest UI
- [ ] Verify description matches current features
- [ ] Ensure screenshots are 1280x800 or 640x400 PNG
- [ ] Update both `en` and `ja` descriptions
- [ ] Copy final assets to Chrome Web Store Developer Dashboard
