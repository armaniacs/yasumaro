# PBI: 検索結果の関連グラフ / タグクラスタ表示

- 親issue: [DEV-86](https://linear.app/armaniacs/issue/DEV-86)
- 領域: A. 知識活用・検索強化
- type: feat / 優先度: 中

## ユーザーストーリー

Yasumaro 利用者として、履歴のタグ共起関係をグラフで見たい。なぜなら、全文検索だけでは関連ページのつながりが見えず、テーマごとの広がりを俯瞰したいから。

## ビジネス価値

蓄積した履歴を「探す」だけでなく「見渡す」体験を提供し、知識活用の入口を増やす。測定: グラフパネルの利用回数 / グラフ経由の履歴クリック率。

## BDD受け入れシナリオ

```gherkin
Scenario: タグ共起グラフが表示される
  Given 複数のタグ付き履歴が記録されている
  When  ダッシュボードでタグクラスタ表示を開く
  Then  タグをノード、共起関係をエッジとしたグラフが描画される
  And   ノードをクリックすると該当履歴が絞り込まれる

Scenario: タグが少ない場合の空状態
  Given タグ付き履歴が存在しない
  When  タグクラスタ表示を開く
  Then  空状態のメッセージが表示される
  And   グラフ描画でエラーにならない
```

## 受け入れ基準

- [ ] tags カラムからタグ共起を集計する
- [ ] グラフノード/エッジを描画する
- [ ] ノードクリックで履歴フィルタと連動する
- [ ] タグ0件で例外を出さない
