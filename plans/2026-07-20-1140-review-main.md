## 重要指摘事項（優先度順）

### [High] CSP `wasm-unsafe-eval` および `'unsafe-inline'` の許可
- 指摘者: Red Team Leader
- 場所: `wxt.config.ts:65`
- 影響: WASM-unsafe-eval と inline-style の許可により、XSS 攻撃による任意のコード実行や、CSS インジェクションを通じた情報漏洩のリスクがある。
- 対処: `'unsafe-inline'` を削除し、スタイルを外部ファイルまたは adoptedStyleSheets に移行。WASM 実行ページのみに限定的な CSP を適用することを検討する。

### [High] マスターパスワード未設定時の API キー暗号化の脆弱性
- 指摘者: Red Team Leader
- 場所: `src/utils/storage/encryptionSession.ts:125-153`
- 影響: マスターパスワード未設定時、暗号化キーが `chrome.storage.local` に平文で保存されており、ローカルストレージへアクセス可能な攻撃者が API キーをすべて復号できる。
- 対処: マスターパスワードを必須にするか、ハードウェアバインドなキー（`chrome.enterprise.platformKeys` 等）を導入して、秘密鍵の平文保存を排除する。

### [High] 4つの独立したマイグレーションシステムの同時実行
- 指摘者: Legacy Bridge Architect, System Architect
- 場所: `src/background/service-worker.ts:82-108`
- 影響: 設定、セッション、SQLiteデータ、OPFS回復の4つのマイグレーションが順序制御なしに並行実行されており、データの整合性が損なわれたり、不整合な設定でマイグレーションが走るリスクがある。
- 対処: `MigrationCoordinator` を導入し、依存関係に基づいたトポロジカル順序での逐次実行を保証する。

### [High] `chrome.storage.local.get()` 無引数呼び出しによるメモリ圧迫
- 指摘者: Ops/Performance
- 場所: `src/background/localMarkdownExportCore.ts:24`
- 影響: ストレージの全データを一度にメモリに読み込んでおり、データ量増加に伴い Service Worker のメモリ圧迫とパフォーマンス低下を招く。
- 対処: 必要なキーのみを明示的に指定して取得し、不要な全件読み込みを排除する。

### [High] `as unknown as MessageHandler` キャストによる型安全性の喪失
- 指摘者: Code Quality
- 場所: `src/background/service-worker.ts:271` 等（全13箇所）
- 影響: 強制的な型キャストにより、ハンドラのシグネチャ不一致がコンパイル時に検出されず、実行時に予期せぬエラーが発生する。
- 対処: `MessageHandler` 型定義を修正し、キャストなしで型が一致するように整備する。

### [Medium] `S-T` 方式の PII サニタイズ失敗時の不適切なフォールバック
- 指摘者: Blue Team Leader
- 場所: `src/utils/piiSanitizer.ts:356`
- 影響: サニタイズ失敗時に `[SANITIZATION_FAILED]` という文字列を返すため、これがそのまま保存され、データにゴミが混入する。
- 対処: エラー時はパイプラインを中断（Abort）させるか、元のテキストをエラーフラグ付きで返し、保存を阻止する。

### [Medium] Service Worker での `setTimeout` 使用によるタスク消失
- 指摘者: Blue Team Leader, Ops/Performance
- 場所: `src/utils/optimisticLock.ts:109` 等
- 影響: SW が終了すると `setTimeout` は消失するため、指数バックオフ中のリトライが実行されず、処理がサイレントに失敗する。
- 対処: `chrome.alarms` に移行し、SW 再起動後もリトライが継続されるようにする。

### [Medium] i18n 日本語翻訳の欠落（30キー以上）
- 指摘者: UX/Frontend
- 場所: `public/_locales/ja/messages.json`
- 影響: UI の一部が英語で表示され、ユーザー体験を損なう。
- 対処: 不足している翻訳キーをすべて日本語化する。
